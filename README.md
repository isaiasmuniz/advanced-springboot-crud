Sobre o Projeto

Este é um projeto de CRUD avançado desenvolvido com Spring Boot, seguindo boas práticas de arquitetura, versionamento de API, tratamento global de exceções, DTOs, testes unitários e documentação automática via Swagger/OpenAPI.

Ele foi construído com foco em boas práticas do mercado, organização limpa e capacidade de evoluir sem quebrar versões anteriores da API.

Tecnologias Utilizadas:

Java 21

Spring Boot

Spring Web

Spring Data JPA


Banco de Dados

MySQL

OpenAPI / Swagger 3

JUnit 5

Mockito

Dozer Mapper

Maven

Testcontainers

Padrões Seguidos:

Controller → Service → Repository

DTO para entrada/saída

Versionamento por /api/v1 e /api/v2

Exception Handler global

Mapeamento separado

Serviço único para toda lógica de negócio
